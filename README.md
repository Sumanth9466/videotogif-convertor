# Video to GIF Converter

This is a simple web tool for converting video files (MP4, MOV, etc.) to animated GIFs right in your browser using [ffmpeg.wasm](https://github.com/ffmpegwasm/ffmpeg.wasm).

## Features
- Converts small video files to GIF
- Runs entirely in your browser (no upload needed)
- Works on most modern desktop browsers

## Usage
1. Open the `index.html` file in your browser **or**
2. Visit the live site (if hosted on GitHub Pages).
3. Select a video and click "Convert to GIF".

## Limitations
- Maximum file size depends on your device and browser memory (recommended <100 MB).
